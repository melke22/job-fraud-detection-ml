## 📖 Overview
With the rapid growth of online job portals and social media platforms, fraudulent job postings have become increasingly common. This project presents a machine learning-based solution to automatically detect fake job advertisements and help users avoid scams.

The system analyzes job postings and classifies them as **genuine** or **fraudulent** using multiple machine learning models and data mining techniques.

---

## 🎯 Objectives
- Detect fake job postings using machine learning  
- Compare performance of multiple classification algorithms  
- Apply Apriori Algorithm for data mining and confidence analysis  
- Build a system that assists users in identifying fraudulent job listings  

---

## 🧠 Technologies Used
- Python  
- Machine Learning (Scikit-learn)  
- Data Mining (Apriori Algorithm)  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

---

## ⚙️ Algorithms Implemented
The project evaluates and compares the following models:

- Decision Tree  
- Logistic Regression  
- Support Vector Machine (SVM)  
- Naïve Bayes  
- Random Forest  
- Multilayer Perceptron (MLP)  

Additionally:  
- Apriori Algorithm is used to determine association rules and confidence values  

---

## 📊 Methodology
1. Data collection and preprocessing  
2. Feature extraction and transformation  
3. Application of Apriori algorithm for pattern mining  
4. Training multiple classification models  
5. Model evaluation and comparison  
6. Selection of the best-performing model
